# ADR-0001: Record architecture decisions

## Status
Accepted

## Context
We need a lightweight, durable record of significant technical choices in LineageIQ.

## Decision
Use Markdown ADRs in `docs/decisions/NNNN-title.md`. Each ADR has Status, Context, Decision,
Consequences.

## Consequences
Decisions are discoverable and reviewable; reviewers can see why, not just what.
